# Radiant Reader 테마 디자인 가이드라인

'Solar Radiant'의 화려함보다 **컨텐츠의 전달력**과 **광고 수익성**에 집중한 새로운 블로그 전용 테마입니다.

## ✍️ 1. 가독성 중심의 타이포그래피 (Typography First)
- **본문 폰트**: 고해상도 디스플레이에 최적화된 `Inter` 또는 `Noto Sans KR` 사용.
- **폰트 크기**: 본문 기본 18px-20px (모바일 16px-18px)로 크게 설정하여 눈의 피로도 감소.
- **행간 (Line-height)**: 1.7 - 1.8로 넉넉하게 설정하여 텍스트의 밀도 조절.
- **문단 구조**: 3~4줄 이상의 긴 문단 지양, 여백(Whitespace)을 적극적으로 활용.

## 💰 2. 애드센스 수익 최적화 레이아웃 (Ad-Placement Strategy)
- **Top Leaderboard**: 네비게이션 바 아래, 본문 시작 전 가장 시선이 많이 머무는 곳에 배치.
- **In-Content Ads**: 본문 1~2번째 문단 사이 및 글 중간 지점에 자연스럽게 삽입.
- **Sticky Anchor**: 모바일 상단 또는 데스크톱 사이드바 하단에 고정형 광고 배치.
- **End of Post**: 글을 다 읽은 후 다음 행동으로 광고 클릭을 유도하는 영역 설정.

## 🎨 3. 비주얼 스타일 및 테마 (Visual Style)
- **배경**: 완전한 화이트(#FFFFFF)보다 눈이 편안한 소프트 그레이(#F8FAFC) 또는 베이지 톤 활용.
- **다크 모드**: 텍스트 대비가 너무 강하지 않은 딥 차콜(#121212) 배경 시스템.
- **포커스 모드 (Focus Mode)**: 읽기 시작하면 불필요한 사이드바와 UI 요소를 투명하게 처리하거나 숨김.

## 🛠 4. 기술적 최적화
- **CLS Zero**: 광고 로딩 시 레이아웃이 밀리는 현상을 방지하기 위해 광고 영역의 최소 높이를 미리 확보.
- **Fast Loading**: 폰트 서브셋팅 및 이미지 Lazy-loading을 통해 LCP 1.2초 이내 달성.
